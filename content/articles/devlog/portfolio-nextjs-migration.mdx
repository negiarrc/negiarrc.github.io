このサイトは Next.js の静的出力を使って GitHub Pages に公開しています。  
アプリ側のロジックを増やさずに運用しやすくするため、記事は `content/articles` 以下に集約しました。

## 運用で意識したポイント

- 記事本文は Markdown/MDX のみで編集
- 記事一覧はフロントマターから自動生成
- `category/slug` を URL とファイルパスで統一
